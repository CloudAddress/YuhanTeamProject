# 게임 제목 : 군생활
------------
## 시스템 구상
- 게임 시작 시 []금액 획득
- 이병부터 시작하며 대장까지 진급이 가능함
- 호봉이 증가할 때마다 [월급]을 수령함
- 호봉을 다 채워 다음 계급으로 [진급]이 가능함
- [진급] 시 [진급비] 수령함
